Website Deliverables

This page outlines the Bootstrap components, dynamic features, and hosting details used in this website.

1. Bootstrap Components Used

  • Layout:Bootstrap containers, rows, and columns are used to structure a responsive page layout.
  • Navbar: Responsive navigation bar with collapsible menu for mobile devices.
  • Images:Bootstrap image utilities are used to ensure images are responsive, styled, and properly aligned.
  • Typography:Bootstrap typography classes are used to control headings, font weights, text size, and readability.
  • Grid System: Containers, rows, and columns used to structure all sections.
  • Cards: Used for projects, experience, and hobbies sections.
  • Badges: Used to display project year and technologies.
  • Accordion: Used in the education section for expandable year details.
  • Buttons: Used for navigation actions and external links.
  • Utilities: Bootstrap utility classes are used for spacing, alignment, and responsive styling.
  • Icons: Used to visually enhance skills and hobbies sections.
  • Modal: Used to display contact information.

2. Dynamic Features Implemented

  • Dark Mode: Website loads in light mode by default with a theme toggle.
  • Responsive Navigation: Mobile-friendly collapsible menu.
  • Accordion Interaction: Expand and collapse education stages.
  • Interactive Buttons: External links open in new tabs.
  • Responsive Layout: Automatically adapts to different screen sizes.
